# 1주차 심화발제 - Simpson의 역설

## 전체 개요
```
[인과추론]
   ↑
   │ (원하는 것: A → B 효과가 있나?)
   │
[A/B 테스트] ─▶ [귀무가설 검정]
   │                ↑
   │                │
   │        통계적으로 유의미한 차이가 있는가?
   │
   └─▶ [Simpson의 역설] (주의: 집계 방식에 따라 결론이 달라질 수 있음)
```

## A/B 테스트의 정의와 구조
A/B 테스트의 정의 : 두 가지 처리 방법, 제품, 절차 중 어느 쪽이 다른 쪽 보다 더 우월하다는 것을 입증하기 위해 실험군을 두 그룹으로 나누어 진행하는 실험

**직관적으로**
```
- 우리는 어떤 조치가 효과가 있는지 알고 싶음
- 예: 새로운 광고 전략이 기존보다 더 나은 효과를 보일까?
  -▶ 이러한 이유로 A/B 테스트를 설계
```

A/B 테스트의 구조
```
A/B 테스트
   └─▶ 귀무가설 검정
          └─▶ 통계적 유의성 판단 (p-value)
```
